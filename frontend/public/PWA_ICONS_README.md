# PWA Icons Required

For production deployment, place the following icon files in this directory:

## Required Files

- `pwa-192x192.png` - 192x192px app icon
- `pwa-512x512.png` - 512x512px app icon
- `apple-touch-icon.png` - 180x180px Apple touch icon
- `favicon.ico` - Standard favicon

## Design Guidelines

- **Theme**: Manufacturing/industrial aesthetic
- **Colors**: Primary blue (#228be6) with white background
- **Icon**: Could feature a factory, gears, checklist, or production line motif
- **Maskable**: The 512x512 icon should work with maskable icon treatment (safe zone in center 80%)

## Generate Icons

You can use tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- Figma/Sketch export at required sizes

## Development Placeholders

For development, simple placeholder icons are acceptable. For production, use professionally designed icons that match the OpenMES brand.
